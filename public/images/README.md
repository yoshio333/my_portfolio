# 画像フォルダ

ここに画像を入れてください。ファイル名の通りのページ・箇所で使われます。

| ファイル名 | 使用箇所 | 向き |
|---|---|---|
| `profile.jpg` | ヒーロー（ポラロイド）・Aboutページ | 縦長 |
| `work-local-tech-camp.jpg` | 実績「Local Tech Camp」カード＆詳細ページ | 横長 |
| `work-roots-youth-hub.jpg` | 実績「Roots Youth Hub」カード＆詳細ページ | 横長 |
| `work-town-hall-2.jpg` | 実績「Town Hall 2.0」カード＆詳細ページ | 横長 |
| `focus-education.jpg` | 重点領域「Educational Ecosystems」詳細ページ上部 | 横長 |
| `focus-youth.jpg` | 重点領域「Youth Mentorship」詳細ページ上部 | 横長 |
| `focus-civic.jpg` | 重点領域「Civic Dialogue」詳細ページ上部 | 横長 |

## 注意
- 拡張子は JPG（`.jpg`）推奨
- サイズ変更不要。Next.js が自動で最適化＆トリミングします
